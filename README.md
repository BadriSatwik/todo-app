# React Todo App

A simple Todo application built with **React (Vite)** that supports full CRUD operations, task filtering, and persistent storage using **localStorage**.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Persist tasks using localStorage

## Tech Stack

- React (Hooks: useState, useEffect)
- Vite
- JavaScript (ES6+)
- CSS

## Project Structure

```

src/
├─ components/
├─ App.jsx
├─ main.jsx

````

## How to Run Locally

```bash
npm install
npm run dev
````

Then open:

```
http://localhost:5173
````
